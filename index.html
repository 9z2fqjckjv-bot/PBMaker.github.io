PBMaker

A GitHub Pages-friendly editor for building a Polandball illustration from a blank white ball with a black outline.

Tools
Actions
How it works
  • It starts with a plain white ball and a bold black outline.
  • Painting is clipped to the inside of the ball, so strokes never spill out.
  • Use “Add eyes” to drop in a classic Polandball face instantly.

Your blank Polandball is ready. Pick a color and start painting.
