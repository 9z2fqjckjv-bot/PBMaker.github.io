# PBMaker.github.io

白地に黒い枠線だけの空白ポーランドボールから、そのままイラストを作成できる GitHub Pages 向けメーカーです。

## できること

- 真っ白 + 黒い枠線の空白ポーランドボールを初期表示
- ブラシで円の内側だけを自由に塗り分け
- 白で塗り戻す簡易消しゴム
- 典型的なポーランドボールの目をワンクリックで追加
- 作成したイラストを PNG で保存

## 使い方

1. GitHub Pages またはローカルで `index.html` を開きます。
2. 色とブラシサイズを選んで、空白のボールに直接描画します。
3. 必要に応じて「目を追加」を押し、最後に「PNG を保存」で書き出します。
