# PBMaker.github.io

白地に黒い枠線だけの空白ポーランドボールから、そのままイラストを作成できる GitHub Pages 向けメーカーです。

## できること

- 真っ白 + 黒い枠線の空白ポーランドボールを初期表示
- ブラシで円の内側だけを自由に塗り分け
- 白で塗り戻す簡易消しゴム
- `template/eyes` の中から目のテンプレートを選んで追加
- `template/balls` と `template/shapes` の画像テンプレートを追加
- 追加したテンプレート画像の移動、サイズ変更、回転、透明度、重なり順の編集
- 作成したイラストを PNG で保存

## 使い方

1. GitHub Pages またはローカルで `index.html` を開きます。
2. 色とブラシサイズを選んで、空白のボールに直接描画します。
3. 必要に応じて「Choose eyes」または Templates から画像を追加します。
4. 追加した画像はキャンバス上でドラッグして移動し、編集パネルでサイズ・回転・透明度を調整します。
5. 最後に「PNG を保存」で書き出します。
